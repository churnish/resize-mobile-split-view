# Resize Mobile Split View

Drag divider line to resize tab groups on mobile.

![Demo of mobile split view resize](assets/demo.gif)

**Touch:** Hold down on a line for a brief moment then move as it lights up.

**Pointer:** Press on a line then move as you hold — just like on desktop.

Tested on iPadOS 26.

## Installation

Until _Resize Mobile Split View_ is [made available](https://github.com/obsidianmd/obsidian-releases/pull/10893) in the plugin directory, to install it:

1. Download and enable the [BRAT](https://churnish.github.io/http-protocol-redirector?r=obsidian://show-plugin?id=obsidian42-brat) plugin.
2. [Install via BRAT](https://churnish.github.io/http-protocol-redirector?r=obsidian://brat?plugin=churnish/resize-mobile-split-view).
3. Select **Add plugin**.

<details><summary>Install manually</summary>

Note: To get updates for _Resize Mobile Split View_, you will have to check for and install them manually.

1. Download `resize-mobile-split-view.zip` in the `Assets` of the [latest release](https://github.com/churnish/resize-mobile-split-view/releases).
2. Open the vault folder in the system file manager.
3. Open your Obsidian configuration folder (`.obsidian` by default, hidden on most OSes).
4. Unzip `resize-mobile-split-view.zip` and place it in the `plugins` folder.
5. Reload plugins or app.
6. Enable _Resize Mobile Split View_ in Obsidian settings → Community plugins → Installed plugins.

</details>

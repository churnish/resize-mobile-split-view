# Resize Mobile Tabs

Drag divider line to resize split view on mobile.

Touch: Hold down the finger on the line for a brief moment then move as it lights up.

Pointer: Press on divider then move as you hold — just like on desktop.

Tested on iPad with iPadOS 26.

## Installation

Until _Resize Mobile Tabs_ is made available in the plugin directory, follow the steps below to install it:

1. Download and enable the [BRAT](https://obsidian.md/plugins?id=obsidian42-brat) plugin.
2. Run _Add a beta plugin for testing_ in the Command palette.
3. Paste https://github.com/churnish/resize-mobile-tabs.
4. Select _Latest version_.
5. Check _Enable after installing the plugin_.
6. Press _Add Plugin_.

<details><summary>Install manually</summary>

1. Download `resize-mobile-tabs.zip` in the `Assets` of the [latest release](https://github.com/churnish/resize-mobile-tabs/releases).
2. Open the vault folder in the system file manager.
3. Open your Obsidian configuration folder (`.obsidian` by default, hidden on most OSes).
4. Unzip `resize-mobile-tabs.zip` and place it in the `plugins` folder.
5. Reload plugins or app.
6. Enable _Resize Mobile Tabs_ in Obsidian settings > Community plugins > Installed plugins.

</details>
